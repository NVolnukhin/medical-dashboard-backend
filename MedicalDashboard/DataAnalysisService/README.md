## Поддерживаемые показатели

- **Pulse** (Пульс): 60-100 уд/мин
- **RespirationRate** (Частота дыхания): 12-20 вдохов/мин
- **Temperature** (Температура): 36.0-37.5°C
- **SystolicPressure** (Давление верхнее): 90-140 мм рт.ст.
- **DiastolicPressure** (Давление нижнее): 60-90 мм рт.ст.
- **Saturation** (Сатурация): 95-100%
- **Weight** (Вес): 30-200 кг
- **Hemoglobin** (Гемоглобин): 120-180 г/л
- **Cholesterol** (Холестерин): 3.0-5.2 ммоль/л
